
What is transportSecurity-1.0?

The transportSecurity-1.0 feature is essentially an alias for the ssl-1.0 feature.
The feature was added to allow customers to configure SSL/TLS without a using a feature
that name that seemed to imply a specific SSL or TLS protocol level.
 
The transportSecurity-1.0 feature does not introduce any functionality beyond what
ssl-1.0 provides.
